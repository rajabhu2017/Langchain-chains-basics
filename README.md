# LangChain Chains - Fundamentals

This repository contains my hands-on practice with the core concepts of **LangChain Chains** using **Google Gemini** and **LangChain**.

The goal of this repository is to understand how different types of chains work and how they can be combined to build AI applications.
## Topics Covered

- Simple Chain
- Sequential Chain
- Parallel Chain
- Conditional Chain
- Prompt Templates
- LangChain Expression Language (LCEL)
- Google Gemini Integration
- Environment Variable Management (.env)

## Tech Stack

- Python
- LangChain
- Google Gemini API
- python-dotenv
